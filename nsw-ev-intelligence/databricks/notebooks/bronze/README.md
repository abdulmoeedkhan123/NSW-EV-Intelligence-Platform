# Bronze Layer - Raw Data Ingestion

Raw data ingestion from various sources into Delta Lake.

## Notebooks

- **ingest_ev.py** - Ingest EV registration, charging stations, usage data
- **ingest_traffic.py** - Ingest traffic flow, congestion, incidents
- **ingest_weather.py** - Ingest weather conditions, forecasts

## Data Sources

- NSW Transport Data
- Bureau of Meteorology
- EV Charging Network APIs
- Traffic Management Systems
