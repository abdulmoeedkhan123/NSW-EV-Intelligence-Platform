# NSW EV Intelligence Platform

Comprehensive data platform for electric vehicle intelligence, traffic analysis, and infrastructure optimization in New South Wales.

## Project Structure

- **databricks/** - Databricks notebooks, jobs, ML models
- **src/** - Source code for ingestion, transformations, AI
- **ui/** - User interface components
- **infra/** - Infrastructure as code (Terraform, Docker)
- **config/** - Configuration files
- **docs/** - Documentation

## Data Architecture (Bronze → Silver → Gold)

**Bronze Layer** - Raw data ingestion
**Silver Layer** - Cleaned and validated data
**Gold Layer** - Business intelligence and analytics

See individual README files in each directory for details.
